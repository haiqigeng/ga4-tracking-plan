# Example Comparison

Use this rule when comparing a client template or historical plan with the
default workbook.

## Preserve When Useful

- familiar sheet names and navigation;
- version history and publication date;
- journey grouping and event-slot layout;
- variable names, display names, value rules, and examples;
- shared GTM/dataLayer protocol rules;
- screenshot placement that helps identify a page or interaction.

## Improve

- replace obsolete or non-GA4 schema with current GA4 decisions;
- consolidate duplicate event definitions;
- separate ecommerce from generic interaction parameters;
- add business questions, analysis use, availability, and ownership where they
  are missing from the underlying plan;
- keep internal reasoning out of visible workbook tabs;
- reduce supporting tabs that do not help analysts or developers.

## Template Diff

Record sheet, column, style, and protected-area changes in the structured plan
when client-template adaptation is required. Do not add a visible template-
provenance section to the workbook unless the client requests it.
